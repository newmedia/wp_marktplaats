== Description ==

Nice dutch targeted plugin which enables your to show the products your are trying to sell on
<http://www.marktplaats.nl> on your own blog!


== Installation ==

Complete installation guide can be found on: <http://projects.newmedia.nl/projects/wpmarktplaats/wiki>

Follow the next steps to set the plugin up for your own usage.

Get the plugin from one of the following places:

Download the tarball from here: http://projects.newmedia.nl/projects/wpmarktplaats/files or
from the wordpress plugin repository.

Or you can run 'on-the-edge' code using our mercurial repository: `hg clone http://hg.newmedia.nl/wp_marktplaats`

*Using the widget*

Now it's time to add the widget to your blog sidebar

1. Go to the admin section of your wordpress blog
2. Select the sidebar where you want to install the plugin
3. Click on 'add' on the wp_marktplaats widget bar to add it to the selected sidebar
4. If needed, you can 'drag and drop' it to an other place in the widget list
Now we can configure the widget

*Configure the widget*

When the widget is in the sidebar, click on the 'edit' link of the widget, now you can change several things:
* Widget title (widget title): This the text presented above the widget on your blog
* Gebruikerscode (usercode): Your unique id on marktplaats. See [[how to get your id]] to get this
* Aantal advertenties (amount of ads): The amount of adverts you want to show maximal per page

Press 'done' when ready.
Press 'save changes' to make wordpress remember the changes you made.
